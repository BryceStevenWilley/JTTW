NAME CONVENTIONS

NO WHITESPACE

Backgrounds: 
  "bg" + Name + ".png/.jpg"
   Example: bgCloud.png

Clouds:
  "cld" + Name + Direction + ".png/.jpg"
   Example: "cldClear.png" is a stationary cloud.
   Example: "cldSunsetDiagR.png" is a cloud that moves up and right.
   
Pedestals: 
  "Pedestal" + Name + ".png/.jpg"
   Example: "Pedestal.png"
   
Rocks:
  "Rock" + # + ".png/.jpg"
   Example: "Rock5.png"

Unique objects:
  Short, meaningful names.
   Example: "BuddhaHand.png"
